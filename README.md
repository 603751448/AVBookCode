### 勘误列表

1、Page-3，Line-8，错误类型--错别字，AudioTrack 写成 "AudiTrack";
2、Page-6，错误类型--描述错误，关于PCM的采样率和比特率换算，常见的比特率并没有44.1Hz，而是44.1KHz，即使是44.1Hz，16位的精度，换算的比特率也不是4410016kb/s的结果;
